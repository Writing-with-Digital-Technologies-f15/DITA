Create a post with Prose.io

These steps will guide you through the process of creating a text file in GitHub using Prose.io

  1. Open your internet browser
  2. Log into github.com
  3. Lot into Prose.io with github authorization
  4. Navigate to appropriate directory/team folder
  5. Click "new file" and compose text in GH markdown
  6. Rename file using format: "LastName-FirstInitial-Wk#"
  7. Click "save" and "commit" to publish to GitHub

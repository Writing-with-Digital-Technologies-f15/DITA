Publishing a post on GitHub through prose.io

This document instructs on how to publish a post to the WRIT 4662W Responses repository on GitHub.com using prose.io

To accomplish this goal you will need to:

  • Have access to a computer with an internet connection
  • Create a GitHub account
  • Authorize prose.io to post to GitHub
  • Write the post and submit it
  1. Navigate to the prose.io webpage.
  2. Click the "Authorize on GitHub" button.
  3. Find "responses" from the list of repositories and click it.
  4. Access your team's folder for that week.
    1. Click on the folder that corresponds with the current week in the semester.
    2. Click on the folder that corresponds with your team's number.
  5. Click the "New File" button.
  6. Write the post.
    1. Type a name for the post in the space at the page's top.
    2. Write in the main body of text.
    3. Use the menu at the top to create headings, link images, create bolded text, etc.
  7. Publish the post to GitHub.
    1. Click the "Save" button.
    2. Write some text describing your post in the smaller text box.
    3. Click the "Commit" button.

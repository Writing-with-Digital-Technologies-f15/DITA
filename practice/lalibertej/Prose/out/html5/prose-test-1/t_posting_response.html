Prose.io is a content editor that can be used to post in Github repositories. This task requires the user to have already set up a Github account.

  1. Enter Prose.io into your browser's URL address bar.
    The Prose.io website will prompt you to "Authorize on Github"
  2. Select the "Authorize on Github" button and log in to your Github account.
    Words
  3. In Prose.io, navigate to the folder you'd like to post inside.

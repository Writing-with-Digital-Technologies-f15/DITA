Creating a Prose.io post

Insert short description here

Prose.io is a content editor that can be used to post in Github repositories. This task requires the user to have already set up a Github account.

  1. Enter Prose.io into your browser's URL address bar.
    The Prose.io website will prompt you to "Authorize on Github"
  2. Select the "Authorize on Github" button and log in to your Github account.
    If you don't have a Github account already, you will need to create one now.
  3. In Prose.io, navigate to the folder (inside Writing-with-Digital-Technologies-f15) where you'd like to post.
  4. Create a new file using the Prose.io "New File" button.
  5. Write post using Github-flavored markdown.
    Remember to delete the dummy text!
  6. Save the file/commit your changes.

Posting a Prose.io Response Post

This is a description of how to create a Prose.io post.

  1. Open Prose.io and select the correct folder.
  2. Create a new file using the button on the right near the top of the page.
  3. Use two hashtags for an h2.
  4. Write your post.
    1. You can use other html elements as well.
  5. Save your file.
    1. You can make changes later and change the file name to record what changes you have made.
  6. Check Github to make sure your formatting turned out correctly.

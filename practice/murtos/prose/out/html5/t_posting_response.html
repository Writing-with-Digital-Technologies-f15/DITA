Posting a Prose.io Response

How to post a prose.io response to GitHub

This task will allow you to write a Markdown post on GitHub using Prose.io.

  1. Login to Github with your account.
  2. Authorize startup of Prose.io.
  3. Select the desired repository and location for your post.
  4. Click on the green box that says “new file”.
  5. Compose your post.
  6. Select filename path from top of page and provide a filename.
  7. Click disk icon on right side of page. Describe the changes you made in the box that appears. Hit commit to save your post.
